# html-svg
This repository provides clear, beginner-friendly demonstrations of the SVG (Scalable Vector Graphics) tag in HTML. It includes practical examples of shapes, paths, gradients, and text, along with accessibility tips and responsive design techniques.


The demo includes examples of:

Circle → positioned at the center with a red fill

Rectangle → blue rectangle drawn with x, y, width, and height attributes

Line → diagonal black line across the canvas

Polygon → green triangle using points

Path → purple cubic Bézier curve with stroke and transparent fill



🔑 Key Concepts Demonstrated
Using the <svg> tag with viewBox for responsive scaling

Applying fill and stroke to style shapes

Combining multiple shapes in one SVG canvas

Understanding coordinate-based drawing (x, y, cx, cy, etc.)



🚀 How to Run
Copy the code into a file named index.html.

Open it in any modern browser.

You’ll see all the shapes rendered together on a 100×100 SVG canvas.
